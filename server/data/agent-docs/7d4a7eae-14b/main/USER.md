## 公司战略（300天后的AI硬件世界）

### 核心理念

**AI大众化的关键：** 不是更复杂的能力，而是更自然的理解与执行

**应用端侧是AI落地的核心：**
- 基座大模型决定 AI 的"智力底线"
- 应用侧竞争决定 AI 的"价值上限"
- 用户最终支付的是"问题被解决"，而非模型参数

### AI时代三级火箭

围绕数据逐级递进：
1. **ASSISTANT** — 历史记忆
2. **COPILOT** — 业务理解  
3. **AGENT** — 环境感知

### AI硬件定位演进

**从"工具"到"独立入口"**
- **工具时代（2022前）：** 功能单一，被动响应，孤立终端
- **入口时代：** AI硬件是AI大脑感知世界的"耳朵"和"眼睛"
  - 听：远场拾音、声源定位、回声消除
  - 看：高清视觉、深度感知、空间建模

### AI硬件闭环架构

```
入口硬件层（耳机/音箱/眼镜）
    ↓
Context层（全时覆盖、Always-on）
    ↓
AI处理层（内容分析、意图提取）
    ↓
Agent执行层（任务触发、执行闭环）
```

### 核心技术壁垒

1. **定制芯片 + 硬件级集成** — 绕过系统限制，拥有最真实的办公原生数据入口
2. **耳机、眼镜、显示器的底层感知逻辑一致性**
3. **软件定义硬件** — 硬件冗余，OTA迭代，而非改件40个月
